# 📊 Global Literacy & Education Trends

## 🔍 Problem Statement
Analyze global literacy, GDP, and education patterns across countries.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- SQL
- Streamlit
- Power BI

## 📊 Features
- Literacy trend analysis
- GDP vs literacy correlation
- Gender gap analysis
- Education index

## 🚀 Dashboard
Interactive Streamlit dashboard with filters and visualizations.

## 📈 Insights
- Higher literacy → higher GDP
- Youth literacy improving
- Gender gap exists in some regions
